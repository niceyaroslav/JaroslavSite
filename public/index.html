Project 1: Simple Weather App
The intent was to build simple weather forecast app using Flask. 

Front-end of this app is written in pure HTML/CSS utilizing Jinja templating language for interactivity. Back-end is written in Python using Flask framework as stated above. It involves API calls to openweathermap.com as well as request handling and data storage in a server-side database (sqlite3). I also deployed it on Heroku. You can try it yourself here!
